Chú ý khi gõ đáp án: dấu ' " ' phải có kí tự '\' ở trước
    VD: "content": "Trong bản ... \" ... \" ...?"

Cách đặt tên file

- Khởi động: 
    + Chung: {câu số}_ques.* 
        VD: 1_ques.png, 2_ques.jpg, 5_ques.mp3, ...
    + Riêng: ts_{số thứ tự thí sinh}_{câu số}_ques.*
        VD: ts_1_1_ques.png, ...
- Tăng tốc: đặt tùy ý
- VCNV: 
    + Hình CNV đặt tùy ý
    + Các hình trong câu hỏi: {câu số}_ques.* 
- Về đích: ts_{số thứ tự thí sinh}_{điểm câu}_{câu số}_ques.*
    VD: ts_1_20_2_ques

Chú ý VCNV:
- Từ khóa hàng ngang: viết liền không cách
